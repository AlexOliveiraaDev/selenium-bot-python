# Automacao Discord

Script para automacao de tarefas no Discord do cria.

## Requisitos
- Python 3+
- `discord.py`

## Como usar
1. Instale as dependencias:
   ```sh
   pip install -r requirements.txt
   ```
2. Configure o `.env` com seu token do bot.
3. Execute o script:
   ```sh
   python bot.py
   ```
